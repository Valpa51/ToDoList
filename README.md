#### RU: Проект - сайт, на котором можно вести свой список дел. Сайт выполнен в японском стиле в преобладающим фиолетовым цветом. Страниц всего одна, на ней же и весь функционал. Первое, что встречает, это основное окно для работы с задачами. Первое поле - название задачи, второе - её описание. Чуть ниже три колонки: Tasks, Progressive, Done. Первая отвечает за "неотфильтрованные" или "не тронутые" задачи, новая задача добавляется автоматически в эту колонку. Остальные две колонки (Progressive, Done) будут заполняться лично вами через перетаскивание задач (клик по задаче левой кнопкой мыши и удерживание, после перенести в нужную колонку). Колонка Progressive - означает в работе. Done - завершённые. Около каждой задачи есть элементы управления: редактирование и удаление. Удаление, соответственно, удаляет задачу. А редактирование задачи производится в изначальном окне для новых задач. В нём появляются данные выбранной вами задачи и вы их меняете. Чтобы закончить редактирование, нужно также нажать на кнопку +. Снизу расположена необходимая информация для работы и связи со мной.

#### EN: A project is a website where you can keep your to-do list. The website is designed in Japanese style with a predominant purple color. There is only one page, and all the functionality is on it. The first thing that comes across is the main task window. The first field is the name of the task, the second is its description. There are three columns below: Tasks, Progressive, Done. The first one is responsible for "unfiltered" or "untouched" tasks, a new task is added automatically to this column. The other two columns (Progressive, Done) will be filled in by you personally by dragging and dropping tasks (left-clicking on an issue and holding it down, then moving it to the appropriate column). The Progressive column means in progress. Done - completed. There are controls next to each task: edit and delete. Deleting, accordingly, deletes the task. And the task is edited in the initial window for new tasks. The data of your chosen task appears in it and you change it. To finish editing, you also need to click on the + button. Below is the necessary information for work and communication with me.
