#### RU: Проект - сайт, на котором можно вести свой список дел. Сайт выполнен в японском стиле в преобладающим фиолетовым цветом. Страниц всего одна, на ней же и весь функционал. Первое, что встречает, это основное окно для работы с задачами. Первое поле - название задачи, второе - её описание. Чуть ниже три колонки: Tasks, Progressive, Done. Первая отвечает за "неотфильтрованные" или "не тронутые" задачи, новая задача добавляется автоматически в эту колонку. Остальные две колонки (Progressive, Done) будут заполняться лично вами через перетаскивание задач (клик по задаче левой кнопкой мыши и удерживание, после перенести в нужную колонку). Колонка Progressive - означает в работе. Done - завершённые. Около каждой задачи есть элементы управления: редактирование и удаление. Удаление, соответственно, удаляет задачу. А редактирование задачи производится в изначальном окне для новых задач. В нём появляются данные выбранной вами задачи и вы их меняете. Чтобы закончить редактирование, нужно также нажать на кнопку +. Снизу расположена необходимая информация для работы и связи со мной.
